# 🔐 OpenClaiming EVM

Canonical EIP-712 payment execution contract for OpenClaiming.

## 🚀 Overview
Implements:
- Delegated payments
- Trustline accounting
- EIP-712 verification (off-chain)

## 📂 Structure
- contracts/OpenClaiming.sol
- scripts/
- test/

## ⚙️ Usage
1. Open line
2. Sign claim off-chain
3. Execute on-chain

## ⚠️ Notes
- Only EOAs sign
- No JSON parsing on-chain
